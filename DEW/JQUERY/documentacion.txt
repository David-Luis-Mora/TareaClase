$(document).ready(): Se utiliza para ejecutar una función cuando el DOM está completamente cargado. Esto asegura que el código dentro de la función no se ejecute hasta que todos los elementos del HTML estén listos.

Documentación: jQuery .ready()
.attr(): Este método se utiliza para obtener o establecer el valor de un atributo en el elemento seleccionado.

Documentación: jQuery .attr()
.text(): Este método se utiliza para obtener o establecer el contenido de texto de los elementos seleccionados.

Documentación: jQuery .text()
.removeClass(): Este método se utiliza para eliminar una o más clases de los elementos seleccionados.

Documentación: jQuery .removeClass()
.click(): Este método se utiliza para asignar una función que se ejecutará cuando el elemento seleccionado sea clicado.

Documentación: jQuery .click()
.append(): Este método se utiliza para insertar contenido al final de los elementos seleccionados.

Documentación: jQuery .append()
.prepend(): Este método se utiliza para insertar contenido al principio de los elementos seleccionados.

Documentación: jQuery .prepend()
.before(): Este método se utiliza para insertar contenido antes de los elementos seleccionados.

Documentación: jQuery .before()
.after(): Este método se utiliza para insertar contenido después de los elementos seleccionados.

Documentación: jQuery .after()
.empty(): Este método se utiliza para eliminar todos los elementos hijos de los elementos seleccionados.

Documentación: jQuery .empty()
.filter(): Este método se utiliza para filtrar los elementos seleccionados en función de una función que se pasa como argumento.

Documentación: jQuery .filter()
.remove(): Este método se utiliza para eliminar los elementos seleccionados del DOM.

Documentación: jQuery .remove()
.toggleClass(): Este método se utiliza para alternar entre añadir y eliminar una clase en los elementos seleccionados.

Documentación: jQuery .toggleClass()

Ejercicio 6

.click(): Se utiliza para asignar una función que se ejecutará cuando el elemento seleccionado sea clicado.

Documentación: jQuery .click()
.show(): Este método se utiliza para mostrar elementos ocultos.

Documentación: jQuery .show()
.hide(): Este método se utiliza para ocultar elementos.

Documentación: jQuery .hide()
.dblclick(): Se utiliza para asignar una función que se ejecutará cuando el elemento seleccionado sea doble clicado.

Documentación: jQuery .dblclick()
.hover(): Este método se utiliza para manejar los eventos de pasar el mouse sobre un elemento, permitiendo ejecutar una función cuando el mouse entra y otra cuando sale.

Documentación: jQuery .hover()
.focus(): Este método se utiliza para asignar una función que se ejecutará cuando el elemento seleccionado reciba el foco.

Documentación: jQuery .focus()
.blur(): Este método se utiliza para asignar una función que se ejecutará cuando el elemento seleccionado pierde el foco.

Documentación: jQuery .blur()
.change(): Este método se utiliza para asignar una función que se ejecutará cuando el valor de un elemento de formulario cambie.

Documentación: jQuery .change()
.submit(): Este método se utiliza para asignar una función que se ejecutará cuando un formulario se envíe.

Documentación: jQuery .submit()
event.preventDefault(): Este método se utiliza para evitar el comportamiento por defecto de un evento (como recargar la página al enviar un formulario).

Documentación: jQuery event.preventDefault()
.keydown(): Se utiliza para asignar una función que se ejecutará cuando se presione una tecla mientras el elemento seleccionado tiene el foco.

Documentación: jQuery .keydown()
.keyup(): Se utiliza para asignar una función que se ejecutará cuando se suelte una tecla mientras el elemento seleccionado tiene el foco.

Documentación: jQuery .keyup()
.keypress(): Se utiliza para asignar una función que se ejecutará cuando se presione una tecla (aunque es menos común y ha sido reemplazada en muchos casos por .keydown() y .keyup()).

Documentación: jQuery .keypress()
